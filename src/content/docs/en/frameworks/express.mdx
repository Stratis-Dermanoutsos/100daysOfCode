***Express*** is a popular back-end web application framework for ***Node.js***,
a ***JavaScript*** runtime that allows developers to run ***JavaScript*** code outside of a web browser.
***Express*** provides a set of tools and libraries for building web applications and APIs using ***Node.js***.
